📄 Technical Documentation

HealthVision AI — Full Documentation

Complete technical reference for the AI-Powered Multi-Health Detection Platform.

1. Introduction

HealthVision AI is a non-invasive, AI-powered health screening platform that leverages deep learning and computer vision to analyze images of eyes, facial skin, and nails for detecting potential health conditions. The platform provides instant screening results without the need for invasive medical procedures.

Project Type: Final-Year Engineering Project
Department: Computer Science & Engineering (Artificial Intelligence)
Institution: Govt. Engineering College, Munger
Supervisor: Dr. Saurabh Suman, Assistant Professor

The platform integrates three specialized deep learning models into a single unified web application, enabling users to upload images or use their device camera for real-time analysis. Each model is built on transfer learning architectures (MobileNetV2 and EfficientNetB0) pre-trained on ImageNet and fine-tuned on curated medical image datasets.

2. Problem Statement

Early detection of health conditions through visual symptoms is critical for timely medical intervention. However, access to dermatological and ophthalmological screening is limited in rural and underserved areas. Many conditions manifest visible symptoms in the eyes, skin, and nails that can be detected through image analysis.

Key Challenges Addressed

  • Jaundice Screening: Elevated bilirubin levels cause yellowing of the sclera — traditionally detected only via blood tests. Our system enables non-invasive screening through sclera image analysis.
  • Skin Disease Identification: Common facial skin conditions (Acne, Eczema, Herpes, Panu, Rosacea) require dermatological expertise. AI can assist in preliminary classification.
  • Nail Disorder Detection: Nail abnormalities can indicate systemic health issues. Early detection through image analysis can prompt timely medical consultation.

3. Objectives

  1. Develop a multi-module AI platform capable of analyzing eye, face, and nail images for disease detection.
  2. Implement transfer learning using MobileNetV2 and EfficientNetB0 architectures for high-accuracy classification with limited training data.
  3. Build an intelligent eye detection pipeline using OpenCV Haar cascades with dual-strategy detection and fallback mechanisms.
  4. Create a responsive web interface supporting both image upload and real-time camera capture with ROI guide overlay.
  5. Ensure privacy by processing all data locally — no images stored or transmitted to third-party services.
  6. Provide detailed disease information alongside predictions for educational and screening purposes.

5. Model Details

5.1 Eye / Jaundice Model (MobileNetV2)

Architecture: MobileNetV2 (Transfer Learning) → GlobalAveragePooling2D → Dense(128, ReLU) → Dropout(0.4) → Dense(1, Sigmoid)
Task: Binary Classification — Jaundice vs Normal
Input: 224×224×3 RGB Image
Parameters: ~2.4 Million (base) + custom head
Layers: 158 total layers
Preprocessing: Rescale pixels to [0, 1] (divide by 255)

The jaundice model analyzes the sclera (white part) of the eye. Before inference, Haar cascade classifiers detect and crop the eye region. The model uses MobileNetV2 pre-trained on ImageNet as a feature extractor, with a custom classification head for binary output (sigmoid activation).

MobileNetV2 GlobalAvgPooling2D Dense 128 (ReLU) Dropout 0.4 Dense 1 (Sigmoid) Binary Cross-Entropy

5.2 Face / Skin Disease Model (EfficientNetB0)

Architecture: EfficientNetB0 (Transfer Learning) → GlobalAveragePooling2D → BatchNormalization → Dense(256, ReLU) → Dropout(0.5) → Dense(5, Softmax)
Task: 5-Class Classification — Acne, Eczema, Herpes, Panu, Rosacea
Input: 224×224×3 RGB Image
Preprocessing: EfficientNet's preprocess_input (scales to [-1, 1])

The face model uses EfficientNetB0 — a compound-scaled CNN architecture that balances width, depth, and resolution for optimal performance. Five common facial skin conditions are classified with softmax output and categorical cross-entropy loss.

EfficientNetB0 GlobalAvgPooling2D BatchNormalization Dense 256 (ReLU) Dropout 0.5 Categorical Cross-Entropy
Class Index Disease Description
0 Acne Clogged follicles causing pimples, blackheads, and whiteheads
1 Eczema (Eksim) Chronic inflammatory condition with dry, itchy, inflamed patches
2 Herpes Viral infection causing painful blisters (HSV-1/HSV-2)
3 Panu Tinea Versicolor — fungal infection with discolored skin patches

5.3 Nail Disease Model (MobileNetV2)

Architecture: MobileNetV2 (Transfer Learning) → GlobalAveragePooling2D → Dense(256, ReLU) → Dropout(0.5) → Dense(3, Softmax)
Task: 3-Class Classification — Healthy, Onychomycosis, Psoriasis
Input: 224×224×3 RGB Image
Preprocessing: Rescale pixels to [0, 1] (divide by 255)

The nail model identifies three categories of nail conditions using MobileNetV2 transfer learning. The model can distinguish between healthy nails, onychomycosis (fungal infection), and psoriasis nail involvement.

MobileNetV2 GlobalAvgPooling2D Dense 256 (ReLU) Dropout 0.5 Categorical Cross-Entropy
Class Index Condition Description

1 Onychomycosis Fungal infection causing discoloration, thickening, and crumbling

7. Training Process

All models use a two-stage training strategy for optimal transfer learning:

Stage 1 — Feature Extraction (Base Frozen)

The pre-trained base model (MobileNetV2 / EfficientNetB0) layers are frozen. Only the custom classification head is trained. This allows the head to learn task-specific features using the pre-trained feature representations.

The top layers of the base model are unfrozen for fine-tuning with a much lower learning rate to adapt pre-trained features to the specific domain.

9. Image Enhancement Pipeline

Before AI inference, every uploaded or captured image passes through an enhancement pipeline to improve prediction accuracy on real-world images taken under varying conditions (lighting, camera quality, angle).

Step 1: Bilateral Denoising

Function: cv2.bilateralFilter(image, d=5, sigmaColor=50, sigmaSpace=50)
Purpose: Removes image noise while preserving edges — critical for maintaining diagnostic features (skin texture, nail ridges, sclera boundaries) while cleaning up camera artifacts.

Step 2: CLAHE Contrast Normalization

Color Space: BGR → LAB (L channel = lightness)
Method: Contrast Limited Adaptive Histogram Equalization (CLAHE)
Parameters: clipLimit=2.0, tileGridSize=(8, 8)
Purpose: Normalizes contrast across different lighting environments (indoor, outdoor, flash, dim). Applied only to the L (lightness) channel to avoid color distortion.

def enhance_image(image): # Step 1: Bilateral filter — removes noise, preserves edges denoised = cv2.bilateralFilter(image, d=5, sigmaColor=50, sigmaSpace=50) # Step 2: CLAHE on L channel for contrast normalization lab = cv2.cvtColor(denoised, cv2.COLOR_BGR2LAB) clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8, 8)) lab[:, :, 0] = clahe.apply(lab[:, :, 0]) enhanced = cv2.cvtColor(lab, cv2.COLOR_LAB2BGR) return enhanced

10. Eye Detection Algorithm

The eye/jaundice module uses a multi-strategy detection pipeline using OpenCV Haar cascades.

Detection Strategies (Priority Order)

Strategy Method Description
1. Face → Eye face_then_eye Detect face first using haarcascade_frontalface_default.xml, then detect eyes within the face ROI using haarcascade_eye.xml. Most accurate method.
2. Direct Eye direct_eye If no face is detected, search for eyes directly in the full image. Works for close-up eye images.
3. Full Image full_image Fallback — use the entire image as-is. Applied when no eye region can be detected.
Key Parameters:
• Image normalized to max 800px before detection
• Histogram equalization on grayscale for better detection
• Face detection: scaleFactor=1.3, minNeighbors=5
• Eye detection: scaleFactor=1.1, minNeighbors=10
• Eye crop padding: 35% of the larger dimension for context

12. Disease Information Database

The platform includes a comprehensive database of disease information returned alongside predictions for educational purposes.

Eye Conditions

👁️ Jaundice Moderate

Description: Jaundice is a condition where the skin and whites of the eyes turn yellow due to elevated bilirubin levels in the blood.

Causes: Liver disease, bile duct obstruction, hemolytic anemia, hepatitis, or certain medications.

Symptoms: Yellowing of skin and eyes, dark urine, pale stools, fatigue, abdominal pain.

Recommendation: Consult a gastroenterologist or hepatologist for blood tests (bilirubin, liver function tests) and imaging studies.

👁️ Normal (Healthy Eye) Healthy

Description: No signs of jaundice detected. The sclera appears normal without yellow discoloration.

Recommendation: Continue regular health checkups. Maintain a balanced diet and stay hydrated.

Skin Conditions

🧑 Acne Mild

Description: Common skin condition caused by clogged hair follicles with oil and dead skin cells.

Causes: Excess sebum production, hormonal changes, bacteria (P. acnes), stress, and diet.

Symptoms: Pimples, blackheads, whiteheads, cysts, redness, and inflammation.

Recommendation: Use gentle cleansers, avoid touching the face, consider topical retinoids or benzoyl peroxide. Consult a dermatologist for persistent acne.

🧑 Eczema Moderate

Description: Chronic inflammatory skin condition causing dry, itchy, and inflamed patches.

Causes: Genetic factors, immune system dysfunction, environmental triggers, allergens, and stress.

Symptoms: Dry, itchy, red, inflamed skin patches, sometimes with weeping or crusting.

Recommendation: Moisturize regularly, avoid triggers, use prescribed topical corticosteroids. Consult a dermatologist.

🧑 Herpes Moderate

Description: Viral infection causing painful blisters or sores, typically around the mouth (HSV-1) or genitals (HSV-2).

Causes: Herpes Simplex Virus (HSV-1 or HSV-2), spread through direct contact.

Symptoms: Painful blisters, tingling before outbreak, swollen lymph nodes, fever.

Recommendation: Antiviral medications (acyclovir, valacyclovir) can reduce severity. Consult a healthcare provider.

🧑 Panu (Tinea Versicolor) Mild

Description: Fungal skin infection caused by Malassezia yeast, leading to discolored patches.

Causes: Overgrowth of Malassezia fungus due to hot/humid weather, oily skin, sweating.

Symptoms: Light or dark patches on skin, mild itching, patches more visible after sun exposure.

Recommendation: Antifungal creams/shampoos (ketoconazole, selenium sulfide). Keep skin dry.

🧑 Rosacea Moderate

Description: Chronic skin condition causing facial redness, visible blood vessels, and small bumps.

Causes: Triggers include sun exposure, hot drinks, spicy food, alcohol, stress.

Symptoms: Facial redness, visible blood vessels, bumps, eye irritation, thickened skin.

Recommendation: Avoid triggers, use gentle skincare, sun protection (SPF 30+). See a dermatologist.

Nail Conditions

💅 Healthy Nail Healthy

Description: No signs of nail disease detected. The nail appears healthy with normal color, texture, and shape.

Recommendation: Continue regular hygiene and health checkups. Keep nails clean and trimmed.

💅 Onychomycosis Moderate

Description: Fungal infection of the nail caused by dermatophytes, yeasts, or molds, leading to discoloration, thickening, and crumbling.

Causes: Fungal organisms (dermatophytes like Trichophyton rubrum), warm/moist environments, damaged nails.

Symptoms: Thickened, discolored (yellow/brown/white) nails, brittle edges, distorted shape, foul odor.

Recommendation: Consult a dermatologist. Treatment includes oral antifungals (terbinafine, itraconazole) or topical antifungals.

💅 Psoriasis Moderate

Description: Nail psoriasis causes pitting, discoloration, thickening, and crumbling of nails.

Causes: Autoimmune condition with overactive immune system accelerating cell turnover.

Symptoms: Nail pitting, oil drop sign, nail thickening, crumbling, separation from nail bed.

Recommendation: Topical treatments (corticosteroids, vitamin D analogs), systemic therapy for severe cases. See a dermatologist.

Extended Nail Disease Database (Information Only)

The platform includes detailed information for 17+ nail conditions. While the trained model classifies into 3 categories (Healthy, Onychomycosis, Psoriasis), the disease database provides educational information for additional conditions that may be added in future versions.

💅 Alopecia Areata (Nail) Moderate

Description: Nail involvement in alopecia areata shows pitting, ridging, or brittleness, indicating autoimmune activity affecting the nail matrix.

Causes: Autoimmune condition where the immune system attacks hair follicles and nail matrix.

Symptoms: Nail pitting, rough texture, ridges, thinning, or complete nail dystrophy alongside hair loss patches.

Recommendation: Consult a dermatologist. Treatment may include corticosteroid injections, topical immunotherapy, or systemic treatments.

💅 Beau's Lines Moderate

Description: Horizontal, transverse depressions across the nail plate caused by temporary disruption of nail growth.

Causes: Severe illness, high fever, chemotherapy, malnutrition, trauma to nail, or systemic conditions like diabetes.

Symptoms: Visible horizontal grooves or indentations running across the nail.

Recommendation: Identify and treat the underlying cause. Consult a physician if recurrent.

💅 Bluish Nail High

Description: Bluish nails (cyanosis) indicate poor oxygen circulation, causing nails to appear blue or purple.

Causes: Respiratory conditions (COPD, pneumonia, asthma), cardiac problems, Raynaud's disease.

Symptoms: Blue or purple discoloration of nails and nail beds, cold fingers, shortness of breath.

Recommendation: Seek medical attention promptly. Can indicate serious cardiovascular or respiratory issues.

💅 Clubbing High

Description: Abnormal widening and rounding of fingertips and nails, often indicating underlying cardiopulmonary disease.

Causes: Lung cancer, heart disease, inflammatory bowel disease, liver cirrhosis.

Symptoms: Widened, rounded fingertips (drumstick appearance), nails curving downward, spongy nail beds.

Recommendation: Seek chest X-ray, echocardiography, and comprehensive medical evaluation.

💅 Koilonychia (Spoon Nails) Moderate

Description: Nails become thin and concave (spoon-shaped), often indicating iron deficiency.

Causes: Iron deficiency anemia, hemochromatosis, hypothyroidism, trauma.

Symptoms: Thin, soft nails with raised edges forming a concave shape, brittle nails.

Recommendation: Check iron levels and CBC. Iron supplementation if deficient.

💅 Leukonychia Mild

Description: White spots or lines on nails, usually caused by minor trauma to the nail matrix.

Causes: Minor nail trauma (most common), zinc deficiency, allergic reaction.

Symptoms: White spots, dots, or lines on nails. Usually painless.

Recommendation: Usually harmless. If persistent, check zinc levels.

💅 Lindsay's Nails Moderate

Description: Half-and-half nails — distinct white proximal and dark distal halves, commonly associated with chronic renal failure.

Causes: Chronic renal failure, azotemia, increased melanin deposition.

Symptoms: Two-toned nails — white proximal half and brown/dark distal half.

Recommendation: Obtain renal function tests and consult a nephrologist urgently.

💅 Muehrcke's Lines Moderate

Description: Paired white transverse bands across the nail bed, associated with low albumin levels.

Causes: Hypoalbuminemia, nephrotic syndrome, liver disease, malnutrition.

Symptoms: Paired horizontal white bands that disappear when nail is pressed.

Recommendation: Check serum albumin levels. Lines resolve when albumin normalizes.

💅 Terry's Nails Moderate

Description: Mostly white nail plate with a narrow dark band at the tip, strongly associated with liver disease.

Causes: Liver cirrhosis, congestive heart failure, diabetes mellitus, aging.

Symptoms: Nails appear mostly white/opaque with a narrow pink-brown band at the distal edge.

Recommendation: Evaluate liver function, cardiac status, and blood glucose.

💅 Red Lunula Moderate

Description: Reddening of the half-moon-shaped area at the nail base, indicating various systemic conditions.

Causes: Cardiac failure, COPD, liver cirrhosis, rheumatoid arthritis, lupus.

Symptoms: Red or erythematous discoloration of the lunula (half-moon area).

Recommendation: Check cardiac, hepatic, and rheumatologic status.

💅 Pale Nail Moderate

Description: Very pale or white nails can indicate anemia, heart failure, liver disease, or malnutrition.

Causes: Anemia, liver cirrhosis, congestive heart failure, diabetes, chronic kidney disease.

Symptoms: Nails appear very pale or white, may have darker band at nail tip.

Recommendation: Check hemoglobin, liver function, kidney function, and heart health.

💅 Darier's Disease Moderate

Description: Genetic skin disorder affecting nails with V-shaped nicking and red/white longitudinal bands.

Causes: Genetic mutation in the ATP2A2 gene, inherited autosomal dominant.

Symptoms: Nail fragility, V-shaped nick at nail tip, red and white bands, warty papules.

Recommendation: Consult a dermatologist for retinoid therapy. Genetic counseling recommended.

💅 Onychogryphosis Mild

Description: Ram's horn nails — thickened, curved, horn-like nails, commonly affecting toenails.

Causes: Aging, peripheral vascular disease, chronic neglect, fungal infection.

Symptoms: Thick, yellowed, curved nails resembling a ram's horn.

Recommendation: Professional nail care by a podiatrist. Regular trimming.

💅 Onycholysis Mild

Description: Painless separation of the nail from the nail bed, starting from the free edge.

Causes: Trauma, fungal infection, psoriasis, thyroid disease, allergic reaction.

Symptoms: Nail lifting from nail bed, white/yellow discoloration at separation point.

Recommendation: Keep nails trimmed short, avoid moisture under nails, treat fungal infections.

14. Frontend Architecture

The frontend is a single-page application (SPA) built with vanilla HTML, CSS, and JavaScript — no frameworks required.

Key Features

  • Module Switching: Eye, Face, and Nail tabs dynamically switch the active prediction endpoint
  • Dual Input Modes: Upload (drag-and-drop + file browser) and Camera (WebRTC live feed with ROI overlay)
  • Camera ROI Guide: Canvas overlay draws a rounded rectangle guide with corner accents, showing users where to position the body part
  • Scan Animation: CSS-animated scan line overlay for visual feedback during analysis
  • Result Rendering: Dynamic result card with severity-colored header, confidence bars, top-k predictions, and disease information accordion
  • AI Assistant Journey: Structured report generation, predefined clinical Q&A, risk-aware recommendations, and doctor checklist output
  • Hospital Finder Cards: State/district-based discovery cards with map links, booking flow, and printable receipt support
  • Chronological Chat Timeline: Hospital cards are appended inline as chat blocks, so every new message appears at the bottom in natural chat order
  • Responsive Design: CSS Grid/Flexbox layouts with media queries for mobile (480px), tablet (768px), and desktop
  • Accessibility: Semantic HTML, ARIA labels, keyboard-navigable interface, high contrast colors

17. Demo Mode

If model files (.h5) are not found on disk, the application automatically enters demo mode — providing simulated predictions so the frontend can be tested without trained models.

Demo Mode Behavior

Module Demo Strategy Details
Eye / Jaundice Heuristic Color Analysis Analyzes sclera yellowness using HSV and LAB color spaces. Detects yellow discoloration via B-channel deviation from neutral (133) in LAB and saturation levels in HSV. Uses multiple fallback masks (primary → permissive → medium) for robustness.
Face / Skin Random Dirichlet Distribution Generates random class probabilities using np.random.dirichlet() to simulate model output for UI testing.
Nail Random Dirichlet Distribution Same random approach as face module for simulated predictions.

Eye Demo — Color Analysis Algorithm

The demo mode for the eye module uses an intelligent heuristic that actually works reasonably well for obvious cases:

  1. Primary Mask: Finds bright, low-saturation, light pixels (sclera-like) — HSV V>160, S<100, LAB L>150
  2. Permissive Mask: For heavily jaundiced eyes with high saturation — HSV V>140, LAB L>120
  3. Medium Mask: Fallback for full-image close-ups — HSV V>160, S<120, LAB L>140
  4. Calculates yellow deviation from neutral B-channel value (133) and saturation score
  5. Combines B-score (60%) + S-score (40%) for final jaundice probability
